# 💪 Gym Schedule & Tracker

A complete React + TypeScript + Vite app to plan your weekly gym schedule and track everything — workouts, sets, reps, weights, body metrics, and goals. All data is saved locally in your browser.

## Features

- 📊 **Dashboard** — Streak, weekly volume, total workouts, latest body weight
- 📅 **Weekly Schedule** — Assign workouts to each day (Mon–Sun) with editable titles
- 🏋️ **Exercise Library** — Add/edit/delete exercises grouped by muscle (Chest, Back, Legs, etc.)
- 🔥 **Live Workout Session** — Timer, checkable sets, add/remove sets on the fly, auto volume calculation
- 📜 **History + Progression** — Per-exercise progression chart (best weight over time)
- ⚖️ **Body Metrics** — Log weight & body fat %, weight trend chart
- 🎯 **Goals** — Track personal goals with deadlines
- 💾 **LocalStorage** persistence — no backend needed
- 📱 Responsive design

## Run

```bash
npm install
npm run dev
```

Open http://localhost:5173
